# fonts（字型）

- 上傳檔案大小限制，故把字型檔 ZIP 壓縮。

- 方案部署前，需把「fonts」資料夾放入手機「rime」資料夾內，且把 Jigmo2.ttf.zip 解壓縮讓 .ttf 在「fonts」第一層，字型才可作用。

- 注意有些解壓縮會出現如「Jigmo2.ttf」檔名之資料夾，此時裝入「部署」程式會崩潰！須要把真正 .ttf 等字型檔放在「fonts」第一層才可！

- (.trime.yaml)「3.3.1 鍵盤檔」沒引用字型，「3.3.2 鍵盤檔」則有。

## Jigmo2.ttf（字雲 Jigmo 字體）

- 中文罕見字（Ext.B, Ext.C, Ext.D, Ext.E, Ext.F）。

- 之前使用「花園明朝體」，更新用新繼承的「字雲」。

- 新的「字雲」：https://kamichikoichi.github.io/jigmo/

- 舊的「花園明朝體」：http://fonts.jp/hanazono/

## TaipeiSansTCBeta-Regular.ttf（台北黑體）

- 一般常用中文。

- 翰字鑄造「台北黑體」：https://sites.google.com/view/jtfoundry

## LucidaGrande.ttc

- 特殊西歐字母。

- Wiki 說明：https://zh.wikipedia.org/zh-tw/Lucida_Grande
