# 3.3.11 鍵盤檔

- 適用[ trime 3.3.11 版](https://github.com/osfans/trime/releases/tag/v3.3.11)

- 該 3.3.11 版實測版本為 Nightly Build 之 [Commit 71d61a0](https://github.com/osfans/trime/actions/runs/28877614405)，使用 android 16。

- 該版輸入翻頁時，多會展開所有選項，無法只在候選列顯示下一頁選項，造成非設想之行為。

- 該版改變「配色」時，如某鍵註解特定顏色，但該「配色」下沒此註解，不會自動補色，會有顏色不顯示之問題。

- 該版無法吃下「click: ""」之設定，該行會空掉，造成鍵盤格式跑掉，但可透過其他設定方法解決。

- 該版測試發現無法吃下：「指定字體」設定。選單中，罕見字會無法顯示。

- 雖有以上 Bug，但該版順暢度穩定度實測 OK！
